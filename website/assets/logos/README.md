# Logo Files for 4Trades.ai Website

## Required Logo

**File name**: `4Trades_Banner_Logo.png` (or .svg)
**Location**: This directory (`website/assets/logos/`)
**Usage**: Navigation bar and footer

**Recommended specs:**
- Format: PNG with transparent background or SVG (preferred)
- Height: 80-120px (will be scaled to 40px in the navbar)
- Width: Auto (maintains aspect ratio)
- Color: Should work well on dark backgrounds (navbar and footer use dark charcoal)

**Supported formats**: PNG, SVG, JPG, WebP

## Current References

The website HTML currently references:
- `assets/logos/4Trades_Banner_Logo.png`

If your logo has a different extension, update the following files:
- Line 21 in `website/index.html` (navbar)
- Line 277 in `website/index.html` (footer)
